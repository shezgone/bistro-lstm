## 판정 공식

매 target 분기 q, **예측 전에 딱 1번** 판정. 모든 입력은 그 시점에 이미 발표된 값만 사용 (vintage-safe).

```
입력:
  flash(q−1) : 직전 분기 GDP 속보치 (발표값)
  vol(q)     : std( flash(q−4) … flash(q−1) )        ← 직전 4분기 변동성
  θ(q)       : median( vol(j) for j < q )             ← 적응형 임계 (확장창)
  S          : 월별 심리지표(ESI), 분기 시작 전월까지 관측분

판정 (위에서부터, 먼저 걸리는 국면 — 순서가 핵심):
  ① REBOUND : flash(q−1) < 0  AND  S 저점통과(마지막 관측월 > 최근 6개월 최저, 저점이 마지막 달 이전)
  ② SHOCK   : vol(q) > θ(q)
  ③ CALM    : 그 외

예측:
  REBOUND → DFM 단독 (보정 OFF)
  SHOCK   → (DFM + XGBoost) / 2
  CALM    → (DFM + RF) / 2
```

- ①을 ②보다 먼저 검사해야 함 — 반등 분기는 대부분 vol도 높아 ②에 걸리면 v1과 동일해짐.
- 예외: 과거 vol 2개 미만 → SHOCK 처리 / 심리 3개월 미만 or flash(q−1) 미발표 → REBOUND 불성립.

## 왜 반등기엔 보정을 끄나

음성장 직후 + 심리 회복 국면은 **학습 표본에 드물어** XGB/RF의 과거 패턴 보정이 잡음이 됨.
반등 6분기에서 DFM 단독 0.621 vs DFM+XGBoost 0.817 — 보정이 체계적으로 해악.
NN(AttnLSTM) 대입도 시험했으나 같은 구간에서 DFM 단독에 미달해 기각 (DFM+seq 0.7411).

심리 조건의 역할 = **연속 추락기 제외**: "직전 분기 음성장"만으로는 2020Q2(−3.3, 추락 지속)가
반등으로 오분류되는데, 심리 저점통과 조건이 이를 정확히 걸러냄. 지표(ESI/CSI) × 관측창(4~12개월)
8개 변형 모두 동일 분류 — 단, 조건이 실질 발동한 것은 2020Q2 한 건이므로 "견고"보다는
"한 번 올바르게 작동"이 정확한 표현.

## 측정 시점 — 32분기 분할

모든 게이트 구성(3-arm/2-arm/DFM+XGB)은 **동일 그리드에서 채점**: 2018Q1–2025Q4 32분기 ×
각 분기 19개 전망주차(w−19~−1) ≈ 627개 예측, flash 기준 RMSE. 기간이 아니라 게이트 규칙만 다름.

3-arm 내부에서 세 모델조합이 발동한 분기:

| arm | 분기 | 성격 |
|---|---|---|
| **REBOUND → DFM 단독** (6) | 2018Q1, 2019Q2, 2020Q3, 2023Q1, 2024Q3, 2025Q2 | 음성장 직후 심리 회복 — COVID 반등(20Q3), 2022 침체 후(23Q1), 계엄 충격 후(25Q2) 등 |
| **SHOCK → (DFM+XGB)/2** (12) | 2018Q2·Q3, 2019Q3·Q4, 2020Q1·Q2·Q4, 2021Q1·Q2, 2024Q4, 2025Q1·Q4 | 직전 4분기 변동성 > 평소 중앙값 |
| **CALM → (DFM+RF)/2** (14) | 2018Q4, 2019Q1, 2021Q3~2022Q4, 2023Q2~2024Q2, 2025Q3 | 나머지 안정 구간 |

v1과의 차이는 REBOUND 6분기뿐 — v1에서는 이들이 SHOCK/CALM으로 흘러 보정이 켜졌고(반등 6Q 0.817),
v2는 보정을 꺼서(0.621) 이 차이가 전체 개선(0.755→0.722)의 전부.

## 성과 (flash w[−19,−1] 평균 RMSE, 2018Q1–2025Q4)

| 게이트 구성 | 전체 32Q | 반등 6Q |
|---|---|---|
| **3-arm (v2)** | **0.7219** | **0.621** |
| 3-arm strict (flash 발표 후 주차만 반등 발동) | 0.7378 | 0.690 |
| 2-arm (v1, 7/2 보고) | 0.7548 | 0.817 |
| DFM+XGBoost (한은 기존 최고) | 0.7650 | 0.815 |

**strict의 의미**: flash(q−1)은 분기 시작 +4주쯤(≈w−12) 발표되므로, 기본 버전이 반등 arm을 분기
전체(19주차)에 적용하는 것은 앞 7주차에 대한 근사임. v1의 vol 판정도 동일한 분기 단위 근사를
쓰므로 **v1 vs v2 비교는 동일 조건**이며, strict(발표 후 주차만 발동)에서도 우위가 유지됨.

## 정직한 캐비엇

1. **arm 선택의 사후성** — "반등→DFM 단독"은 반등 6분기 성적을 보고 고른 구조 (v1의 구조 선택보다 한 단계 더 쌓인 선택편향).
2. **소표본** — 반등 분기별 4승 2패(2023Q1·2024Q3 소폭 열세). 한 분기가 뒤집으면 흔들림.
3. **DM 유의성 미검증** — 한은 정식 보고 전 필수.
4. LLM 반등 arm은 보류 — 무오염 검증 전이며, 기준선이 "반등기 DFM 단독"으로 상향됨 (2025Q2 오염 표본에선 DFM+LLM 0.096 < DFM 0.176).
